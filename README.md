# 🍰 Cake Shop Website

A professional and responsive Cake Shop Website developed using **HTML, CSS.  
This project represents a modern bakery website where users can explore cakes, view details, and enjoy a smooth UI experience.

---

## 🚀 Features

- Responsive design (Mobile, Tablet, Desktop)
- Attractive UI with modern layout
- Cake product listing section
- Smooth CSS animations and hover effects
- Clean and structured code
- User-friendly navigation

---

## 🛠️ Technologies Used

- HTML5  
- CSS3
